## üíª Getting started

### Requirements

- You need to install both [Expo](https://expo.io) and [Yarn](https://yarnpkg.com/) to run this project.

**Clone the project and access the folder**

```bash
$ git clone https://github.com/marciobahia/aircnc.git && cd aircnc
```
